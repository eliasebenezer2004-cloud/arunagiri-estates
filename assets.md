## 9. Technical Specifications

| Property | Requirement |
|----------|-------------|
| Image Format | JPEG for photos, PNG for logos/graphics, WebP as modern alternative |
| Max File Size | 200KB per image (after optimization) |
| Recommended Dimensions | Hero: 1920×1080px, Cards: 800×600px, Gallery: 1200×800px, Thumbnails: 400×300px |
| Lazy Loading | Yes — all images below the fold must use `loading="lazy"` |
| Alt Text | Every image must have descriptive alt text (Tamil + English where relevant) |
| CDN | Consider using Cloudinary or Imgix for automated optimization |
| Compression | Use TinyPNG or Squoosh before uploading |
| Responsive | Serve different sizes via `srcset` for mobile/tablet/desktop |

4. **Ongoing:** Replace team stock photos with real team photos
